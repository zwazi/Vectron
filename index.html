Help & Reference

Tools

ToolKeyDescription
Select1Click an object to select it. Drag on empty space to box-select. Click a selected object to deselect. Del deletes the selection.
Navigation2Click and drag to pan the view. Alternatively use middle-click drag or arrow keys at any time.
Wall3Open the Wall Tool window to choose freeform walls, rectangles, circles, arcs, ellipses, or text walls. For shape tools, click the requested points, set the wall count if prompted, then press Generate Walls. Text walls use a typed string, a two-corner box, and Submit. Double-click or press Shift+W to finish freeform walls. Press Esc to cancel.
Zone4Choose a zone type in the Zone Tool window, then click to place. +/= increase radius; -/_ decrease radius. Shift+Z cycles zone types.
Spawn5Click to place a spawn point. Move the cursor to set the spawn direction, then click again to confirm.
Snap Toggle6Toggles grid snapping on/off. The lock/unlock icon in the toolbar reflects the current state.
Wall Split7Click on a wall to split it at the nearest point. Hover over the wall to see the split preview.
Wall Join8Click the endpoint of one wall, then the endpoint of another to merge them into a single wall.
Vertex Move0Click to select a wall, then drag its vertex dots to reposition them. Del deletes the selected vertex.

Top Bar Fields

NameMap name included in the exported file
AuthorAuthor name included in the exported file
CategoryMap category (e.g. racing, sumo)
RevisionMap version number. Servers and clients cache maps; increment this each time you update so the new version is downloaded.
DTDThe DTD file the map is validated against. Common values: sty.dtd, map-0.2.8.dtd, aacockpit-0.1.dtd. Type freely or pick from the dropdown.
Axes ☐Number of axes (turning directions). axes = 360 / angle. Check the box to include this in the exported map file.

Settings Menu Tabs

Map SettingsSettings tags embedded in the map file (not your server config).
Adjust MapScale, rotate, translate, or center all objects at once. "Center over 0,0" moves all objects so their group center lands on the world origin.
ConfigureTheme, info bar, action history, grid layout/colors, and keybinds.

Toolbar Buttons

XML EditorView and edit the map XML directly. Changes are applied immediately.
New MapClears all objects and resets the map. Unsaved changes are lost.
ImportLoad an .aamap.xml file from disk.
ExportSave the current map as an .aamap.xml file.
Action HistoryToggle the action history panel (also accessible from Configure).
SettingsOpen/close this settings window.

Vectron 1.1

A map editor for Armagetron Advanced. It was originally written in 2008 by Carlo Veneziano, then re-written 2010 as a Flash-based map editor, then re-written in 2014 by Tristan in Javascript. (David Dubois is also credited.)

The project was pretty much inactive until around 2017, when someone requested it be added to my (Armanelgtron) site. However, it still essentially remained with very few changes until 2021 when I finally decided to attempt to fix some annoyances and improve upon it.


Source code available at https://gitlab.com/Armanelgtron/Vectron/.
